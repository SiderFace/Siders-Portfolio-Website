MyFlix DB [React]

CLICK HERE to go to 'MyFlix DB' hosted site

User Experience :

This is a movie database (in the spirit of iMDB and Rotten Tomatoes) the provides user with a database of movies to explore. Each movie has a brief synapse and descriptions of the genre they are in and the director of the movie. User's will be prompted to register with the site, at which time a Profile page will be generated for them. They can also un-register any time they desire. When exploring the movies, they can add any of the movies to their own personal "Favorites" list, which will be displayed on their Profile page, and can be removed at any time.

Features :

This app features Anonymous user registration and token authorization. It is a single-page Application (SPA) with several co-dependant components working together. THe search field filters movies, by title, as they type characters into the search field. The rendered movie database is supplied by API calls to a Mongo Atlas database hosted by Heroku. A profile page is offered to each unique user respectively, where they can compile their own array of movies.

Meet App [Testing]

CLICK HERE to go find events at the 'Meet App' hosted site

User Experience :

This app provides a list of coding events all around the world. The user can use the Search bar to filter the events by city, as well as control the number of events that will be displayed at one time. Above the event list are two graphs giving the user data on the frequency of events by city, as well as the frequency of events by their subject matter, on their filtered list of events. Users will have to log in with their google-gmail credentials, which allows them to seamlessly add events they find to their own personal google calender, where they can benefit from the various benefits therein.

Features :

This app makes for a great playground and learning experience for working with a variety of different testing tools and methods. It sources events from an external public database. The list is rendered based on user-selected filter by city. The charts at the top represent what is currently rendered on the filter list, displaying the frequency of those events by city and event subject matter.

The rendered movie database is supplied by API calls to a Mongo Atlas database hosted by Heroku. Chat App [React Native]

CLICK HERE to go to instructions on running the app locally on your machine

User Experience :

Users of this mobile device Chat App can select a username and background color, and then immedately start chatting with any users conencted to the app. They are able to share messages containing text, camera photos, stored images, and their GPS-generated location. Users can expect chat history to be saved at all times, and remain in the chat upon future returns, or after losing internet connectivity. While disconnected users can still see the chat contents that were sent and received up until the point of losing their connection.

Features :

The app generates a chat icon for each user based on teh initials of their selected username. The background of the chat window is generated with teh color selected by the user. File sharing is mad epossible by the firestore cloud storage that hosts the shared access point where all connected users can send and receive messages, images, and locations. Asynchronous data storage allows for any chat data that was sent before a user loses internet connectivity to remain visible to the user. The chat is equipped to utilize the users' device GPS capabilities to share their location with other users. Users will always be prompted to give permission to the app to access their camera, file (images) storage, and location (GPS) to ensure trust and privacy.

To-do List App [JS: jQuery]

CLICK HERE to go to the 'To-Do List App' hosted site

User Experience :

Users of the To-do List App can utilize this classic tool to make a log of items that can be crossed off or removed at their discretion. Users simply type in the item they want to add to the list into the given text input field, and click "Add". They can click the "X" beside each listed item to remove it, or double-click the item to have it remain listed but crossed off. A simple yet great way to keep track of things otherwise easy to forget.

Features :

The App genertes list items, and removes them upon command. The text input field gives suggestions of items to add, as the user types, based on past items the user has added in previous lists.
